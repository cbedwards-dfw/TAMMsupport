# Reads chunk of 2A_CUnmrkd sheet

Reads chunk of 2A_CUnmrkd sheet

## Usage

``` r
chunk_read_2A_CUnmrkd(full_sheet, start_col, end_col, table_name)
```

## Arguments

- full_sheet:

  entire raw spreadsheet

- start_col:

  Letter of the column the chunk starts on

- end_col:

  Letter of the column the chunk ends on

- table_name:

  name of table
