# Coho fishery information

Mapping of fishery_id to fishery name for Coho salmon, taken from the
`Fishery` table of the Coho FRAM database.

## Usage

``` r
fishery_coho_fram
```

## Format

A data frame with 198 rows and 5 columns:

- species:

  Species name

- version_number:

- fishery_id:

  Coho fishery id number in FRAM

- fishery_name:

  Coho fishery name in FRAM

- fishery_title:

  consistent and more human readable version of `fishery_name`

## Source
