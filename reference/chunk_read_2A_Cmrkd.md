# Read in chunks of the 2A_Cmrkd sheet

Read in chunks of the 2A_Cmrkd sheet

## Usage

``` r
chunk_read_2A_Cmrkd(tamm_filepath, start_col, end_col, table_name)
```

## Arguments

- tamm_filepath:

  tame file path

- start_col:

  Letter of the column the chunk starts on

- end_col:

  Letter of the column the chunk ends on

- table_name:

  name of table
