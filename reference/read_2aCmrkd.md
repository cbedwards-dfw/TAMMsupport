# Reads sheet of 2A_Cmrkd

Reads sheet of 2A_Cmrkd

## Usage

``` r
read_2aCmrkd(tamm_filepath, stock_cleanup = TRUE)
```

## Arguments

- tamm_filepath:

  path to file

- stock_cleanup:

  Do cleanup of capitalization?

## Value

a list, with \$aeq (the main parts of each table) and \$er (the very
bottom section)
