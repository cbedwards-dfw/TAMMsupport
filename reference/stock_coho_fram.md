# Coho stock information

Mapping of stock_id to stock name for Coho salmon, taken from the
`Stock` table of the Coho FRAM database.

## Usage

``` r
stock_coho_fram
```

## Format

A data frame with 78 rows and 7 columns:

- species:

  Species name

- stock_version:

- stock_id:

  Coho stock id number in FRAM

- production_region_number:

- management_unit_number:

- stock_name:

  Coho stock name in FRAM

- stock_long_name:

  `stock_name` but more human readable
